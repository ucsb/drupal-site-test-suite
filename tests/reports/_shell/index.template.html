About this report

Why this matters

This report summarizes what automated tests flagged on rendered site pages and in the custom themes, modules, and profiles shipped with the site. Automated testing provides only partial accessibility coverage, so manual testing with screen readers, keyboard-only navigation, and people with disabilities is still necessary.

Treat this report as a shared checkpoint, not a scoreboard. Findings are grouped by user impact to show who may be affected, and by severity to support prioritization. Each finding includes technical details such as the rule ID, file path or selector, and the original tool message for remediation.
